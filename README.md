# Nocross Online Replica

This project is a replication of the website `https://nocross-online.com/`.

## Overview
It recreates the structure, content, and design of the original corporate website for 株式会社ノクロス (Nocross Inc.).

## Structure
- `index.html`: Main content structure using semantic HTML5.
- `css/style.css`: Styling to match the clean, corporate aesthetic.
- `images/`: Contains the assets used on the page.

## Running Locally
You can serve this project using any static file server.
Example with Python:
```bash
python3 -m http.server 3000
```
